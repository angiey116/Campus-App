# Campus Life Super App

## Description
This project is a multi-page responsive web application designed to help students find campus events, dining info, and useful campus resources.

## Purpose
The purpose of this project is to practice building a responsive website using HTML, CSS, Bootstrap, JavaScript, and GitHub Pages.

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
- GitHub Pages
- Visual Studio Code

## Features <!--List the main features of the project-->
- Home page
- Events page
- Dining page
- Internal navigation
- Interactive buttons with JavaScript
- Responsive layout
- Bootstrap styling

## Setup Instructions <!--Explain how to open and run the project-->
1. Clone the repository
2. Open the project in VS Code
3. Open index.html in a browser

## Deployment <!--Explain where the project is deployed-->
This project is deployed using Github Pages.
